This is a Simple API Documentation

Users Routes

# Request Type Route Description Header Content
1 POST /users Create or Sign Up new users None
2 POST /users/login Login Users Key-Authorization Value-Bearer JWTTokenVALUE
3 POST /users/logout Logout Users removing current token Key-Authorization Value-Bearer JWTTokenVALUE
4 POST /users/logoutAll Logout Users remiving all tokens Key-Authorization Value-Bearer JWTTokenVALUE
5 GET /users/me Fetch User Profile Key-Authorization Value-Bearer JWTTokenVALUE
6 PATCH /users/me Update Users Profile Details Key-Authorization Value-Bearer JWTTokenVALUE
7 DELETE /users/me Remove Users Account Key-Authorization Value-Bearer JWTTokenVALUE
8 POST /users/me/avatar Upload Users Profile Image Key-Authorization Value-Bearer JWTTokenVALUE
9 DELETE /users/me/avatar Remove Users Profile Image Key-Authorization Value-Bearer JWTTokenVALUE
10 GET /users/id/avatar Fetch Users Profile Image None

Tasks Routes

# Request Type Route Description Header Content
1 GET /tasks Fetch All the tasks of the current user Key-Authorization Value-Bearer JWTTokenVALUE
1.1 GET /tasks?completed=true Fetch all the completed tasks of the current user. Use false for incompleted task Key-Authorization Value-Bearer JWTTokenVALUE
1.2 GET /tasks?limit=10&skip=20 Fetch limited tasks of the current user Key-Authorization Value-Bearer JWTTokenVALUE
1.3 GET /tasks?sortBy=createdAt:desc Fetch tasks of the current user in ascending (asc) or descending (desc) order. Key-Authorization Value-Bearer JWTTokenVALUE
2 POST /tasks Add new tasks Key-Authorization Value-Bearer JWTTokenVALUE
3 GET /tasks/id Fetch particular task of the current user Key-Authorization Value-Bearer JWTTokenVALUE
4 PATCH /tasks/id Update a particular task Key-Authorization Value-Bearer JWTTokenVALUE
5 DELETE /tasks/id Delete a particular task Key-Authorization Value-Bearer JWTTokenVALUE
